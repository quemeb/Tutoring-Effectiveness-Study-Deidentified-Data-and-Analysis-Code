# Tutoring-Effectiveness-Study-Deidentified-Data-and-Analysis-Code
De-identified CBE tutoring datasets and Stata code for replication.


This repository contains de-identified datasets and Stata analysis code supporting the study “How Tutoring Supports Progress in Fully Online, Self-Paced Learning: Evidence from a Competency-Based College.”
It includes anonymized records for Quarter 1 (Jan–Mar 2025), Quarter 2 (Apr–Jun 2025), and the combined six-month sample, along with the Stata .do file used to reproduce all descriptive and regression analyses.
